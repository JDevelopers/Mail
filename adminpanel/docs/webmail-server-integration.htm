Server Integration

Enable Integration
Allows managing accounts on AfterLogic XMail Server from AfterLogic WebMail Lite.
Path to Server
Path to the MailRoot folder of AfterLogic XMail Server in your system, for instance C:\Program Files\XMail\MailRoot\.
Server Host
IP address or hostname where AfterLogic XMail Server is installed and running.
Allow users to manage accounts on AfterLogic XMail Server
If a user adds or removes a linked account in his primary account settings and domain part of this account matches any of your domains hosted by AfterLogic XMail Server, this account will be added/removed on AfterLogic XMail Server.
